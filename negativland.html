Negativland

This joint project was to create and promote an event for the band Negativeland. We chose to host a concert at Area 51, inside Warehouse 51, which has unknown uses. We created a special package that could be mailed to a select few, and it included: A small model rocket with hidden cargo (aka sample band music), some moon men, and a set of coasters. There's also a bulk mailable post card that doubles as a ticket for the event, an event ticket to be purchased at the door, and a business card for the band manager.
