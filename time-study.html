A Study of Time

What is time? This is a fairly common question that many people ask. We assume that time is linear, since that's how we experience it. Is it really though? This project is a brief exploration into some of these questions and even a few others. I built the project around the idea of a type of interface that a user could look through to find different answers to questions about time. With each question providing an short animated response. There's a subtle tick-tock sound in the background, almsost as an ominious reminder that all the while, time is ticking by.
